### Run the Server ###

First, Install Node
Second, Install rethinkdb

Download mind map server

Install dependencies and create symlnk to app assets


```
#!bash

 $ npm install -d
 $ ln -s client /path/to/mind-map-app
 $ ln -s landing /path/to/mind-map-landing
 $ node index.js
```


browse to http://localhost:3000
